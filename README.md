# 🌾 AgriSim: AI-Powered Crop & Farming Simulator

## 📌 Problem

Farmers face difficulty in predicting crop yield, profit, and risk due to changing climate and soil conditions.

## 💡 Solution

AgriSim is an AI-powered simulator that predicts yield, revenue, profit, and risk based on farm inputs like soil type, weather, fertilizer, and irrigation.

## 🚀 Features

- Farm configuration (location, soil, crop)

- Real-time yield prediction

- Profit & cost analysis

- Risk assessment

- Interactive dashboard

## 🛠 Tech Stack

- Frontend: React

- Backend: Python (ML Models)

- Visualization: Charts & Dashboard UI

## ▶️ How to Run

1. Clone the repo

2. Install dependencies

3. Run the project

## 🌟 USP

Real-time AI-powered farming decision tool combining multiple environmental and input factors.
